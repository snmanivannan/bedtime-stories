# DreamTales AI - Personalized Bedtime Stories

Create magical, personalized bedtime stories for your child with AI-generated narratives and soothing voice narration.

## Features

- **Personalized Stories**: Enter your child's name, age, and interests to generate unique stories
- **AI-Powered**: Stories generated by Google Gemini AI with age-appropriate vocabulary
- **Voice Narration**: Soothing bedtime narration powered by ElevenLabs TTS
- **Story Library**: Save and replay favorite stories (stored in browser localStorage)
- **Beautiful UI**: Child-friendly design with gentle animations
- **Mobile Responsive**: Works great on all devices

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: TailwindCSS + Shadcn/ui
- **AI**: Google Gemini API
- **Text-to-Speech**: ElevenLabs API
- **Animations**: Framer Motion

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm or yarn
- Google Gemini API key
- ElevenLabs API key

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/dreamtales-ai.git
cd dreamtales-ai
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
```bash
cp .env.example .env.local
```

4. Add your API keys to `.env.local`:
```env
GEMINI_API_KEY=your_gemini_api_key_here
ELEVENLABS_API_KEY=your_elevenlabs_api_key_here
```

### Getting API Keys

#### Google Gemini API
1. Go to [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy the key to your `.env.local` file

#### ElevenLabs API
1. Go to [ElevenLabs](https://elevenlabs.io)
2. Create an account (free tier available)
3. Go to Profile Settings â†’ API Keys
4. Copy your API key to your `.env.local` file

### Running Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

1. **Create a Story**
   - Enter your child's name
   - Select their age (1-12 years)
   - Choose interests (dinosaurs, space, fairies, etc.)
   - Pick a moral lesson
   - Select story length (5/10/15 minutes)
   - Click "Create Bedtime Story"

2. **Listen to the Story**
   - View the generated story text
   - Click "Generate Narration" for voice audio
   - Use audio player controls to play/pause
   - Download audio for offline listening

3. **Story Library**
   - All stories are saved automatically
   - Browse past stories in the library
   - Search by title, name, or interests
   - Replay or delete old stories

## API Limits

### Google Gemini (Free Tier)
- 60 requests per minute
- 1 million tokens per minute

### ElevenLabs (Free Tier)
- 10,000 characters per month
- ~10-15 stories depending on length

## Customization

### Changing the Voice
Edit `lib/elevenlabs.ts` to use a different voice:
```typescript
export const VOICE_OPTIONS = {
  bella: "EXAVITQu4vr4xnSDxMaL",  // Default
  rachel: "21m00Tcm4TlvDq8ikWAM",
  adam: "pNInz6obpgDQGcFmaJgB",
  josh: "TxGEqnHWrfWFTfGW9XjX",
};
```
